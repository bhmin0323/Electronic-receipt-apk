# Electronic-receipt

## 프로젝트 개요
본 시스템은 기존 POS 시스템에 확장 가능한 전자영수증 시스템으로서, 사용자가 종이 영수증 대신 웹 브라우저와 모바일 어플리케이션을 통해 영수증을 열람할 수 있는 시스템이다.
